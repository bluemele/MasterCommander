How It Works

Three steps from installation to your first WhatsApp alert.

01

Install SignalK

SignalK is free, open-source software that translates your marine electronics into a standard JSON format. Runs on the same Mac Mini as Commander.

signalk-server auto-detects NMEA 2000 devices via USB gateway.

02

Run Commander

Commander connects to SignalK, auto-discovers every sensor on your boat, and starts monitoring. Twin engines? Two battery banks? It figures it out.

node commander.js — scan the QR code, done.

03

Get WhatsApp Alerts

Battery low? Engine overheating? Bilge pump cycling? You get a WhatsApp message instantly. Send commands back to check any system on board.

Text status anytime for a full report.

Features

Everything you need to monitor your boat from anywhere.

🔍

Auto-Discovery

Commander scans SignalK and learns what sensors YOUR boat has. No configuration files, no device lists. Adapts to any vessel automatically.

🚨

Smart Alerts

Rule-based alert engine — no AI, no hallucinations. Pure code monitoring thresholds 24/7. Battery, engines, bilge, anchor, depth, tanks.

⚡

Quick Commands

Instant responses for status, engines, battery, tanks, wind, anchor, and position. Pre-built templates, no AI latency.

🧠

Local AI Brain

Qwen 14B runs on-device via Ollama — fully offline. Ask questions in plain English. Optionally route to Claude via Starlink for deeper analysis.

📶

Offline-First

Designed to work with zero internet. Monitoring, alerts, and local AI all run without connectivity. Cloud is an enhancement, not a dependency.

⚓

Anchor Watch

Set your swing radius and Commander monitors GPS drift continuously. If your anchor drags, you know immediately via WhatsApp.

🎮

Built-In Simulator

Test without a boat. Simulate catamaran, monohull, or powerboat profiles. Switch between motoring, sailing, anchored, and alarm scenarios on the fly.

🌍

Fleet Management

Multiple Commanders report to Master — the optional cloud layer. Fleet dashboard, maintenance tracking, cross-boat analytics for charter companies.

Automatic Alerts

Alerts fire automatically via WhatsApp. No need to ask. All thresholds are configurable.

Alert Trigger Requires
🔋 Battery Low SOC < 20% Any battery on SignalK
🔋 Battery Critical SOC < 10% Any battery on SignalK
🌡️ Engine Overheat Coolant > 95°C Engine on NMEA 2000
🛢️ Low Oil Pressure < 25 PSI while running Engine on NMEA 2000
🚰 Bilge Pump > 6 cycles in 30 min Bilge pump counter
⚓ Anchor Drag GPS drift > set radius GPS + anchor watch
🌊 Shallow Water Depth < configured min Depth sounder
⛽ Low Fuel < 15% Tank sensors
💧 Low Water < 15% Tank sensors
